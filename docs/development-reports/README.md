# Rapports de développement

- Objet: centraliser les comptes-rendus techniques et opérationnels réalisés sur BantuDelice.
- Convention: un fichier par intervention majeure.
- Format recommandé: `YYYY-MM-DD-sujet.md`.
- Contenu minimal:
  - titre
  - date
  - contexte
  - actions réalisées
  - validations
  - points ouverts

Exemple de chemin utilisé par l'admin ops/dev:

- `docs/development-reports/2026-03-30-mobile-money-checkout-hardening.md`
