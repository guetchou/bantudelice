# BantuDelice

Plateforme multi-service Laravel pour:
- livraison de repas
- livraison de colis
- transport / taxi
- paiements Mobile Money

Le point d'entrée documentaire pour tout nouveau développeur est:
- [`docs/DEVELOPER_GUIDE.md`](./docs/DEVELOPER_GUIDE.md)

Ce guide couvre:
- architecture réelle du projet
- flux métier critiques
- fichiers importants
- commandes utiles
- conventions de déploiement
- zones sensibles à ne pas casser

Documentation conservée en complément:
- `docs/colis/` pour le module colis
- `docs/INCIDENT_2026-03-19_DATABASE_OUTAGE.md` pour l'incident base de données

Tout document historique ou de migration devenu obsolète a été retiré pour réduire le bruit.
